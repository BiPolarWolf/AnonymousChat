# Redis функции в проекте

## Используемые функции

### redis.from_url()
**Что делает:** Создает подключение к Redis серверу  
**Вход:** 
- `url` (str) - адрес Redis сервера, например "redis://localhost:6379"
- `decode_responses` (bool) - если True, автоматически декодирует ответы в строки

### r.setex()
**Что делает:** Сохраняет значение с автоматическим удалением через указанное время  
**Вход:**
- `key` (str) - ключ для сохранения
- `time` (int) - время жизни в секундах
- `value` (str) - значение для сохранения

### r.get()
**Что делает:** Получает значение по ключу  
**Вход:**
- `key` (str) - ключ для получения значения  
**Выход:** строка или None если ключ не найден

### r.keys()
**Что делает:** Находит все ключи по паттерну  
**Вход:**
- `pattern` (str) - паттерн поиска, например "chat_msg:general:*"  
**Выход:** список найденных ключей

## Другие полезные функции

### r.set()
**Что делает:** Сохраняет значение без TTL (живет вечно)  
**Вход:**
- `key` (str) - ключ
- `value` (str) - значение

### r.delete()
**Что делает:** Удаляет один или несколько ключей  
**Вход:**
- `*keys` (str) - один или несколько ключей для удаления

### r.exists()
**Что делает:** Проверяет существование ключа  
**Вход:**
- `key` (str) - ключ для проверки  
**Выход:** 1 если существует, 0 если нет

### r.expire()
**Что делает:** Устанавливает TTL для существующего ключа  
**Вход:**
- `key` (str) - ключ
- `seconds` (int) - время жизни в секундах

### r.ttl()
**Что делает:** Получает оставшееся время жизни ключа  
**Вход:**
- `key` (str) - ключ  
**Выход:** секунды до удаления, -1 если без TTL, -2 если ключ не существует

### r.incr() / r.decr()
**Что делает:** Увеличивает/уменьшает числовое значение на 1  
**Вход:**
- `key` (str) - ключ  
**Выход:** новое значение

### r.lpush() / r.rpush()
**Что делает:** Добавляет элемент в начало/конец списка  
**Вход:**
- `key` (str) - ключ списка
- `*values` - значения для добавления

### r.lrange()
**Что делает:** Получает элементы списка по диапазону  
**Вход:**
- `key` (str) - ключ списка
- `start` (int) - начальный индекс
- `end` (int) - конечный индекс (-1 для конца)  
**Выход:** список элементов

### r.ltrim()
**Что делает:** Обрезает список, оставляя только указанный диапазон  
**Вход:**
- `key` (str) - ключ списка
- `start` (int) - начальный индекс
- `end` (int) - конечный индекс

### r.hset() / r.hget()
**Что делает:** Сохраняет/получает поле в hash-структуре  
**Вход:**
- `key` (str) - ключ hash
- `field` (str) - имя поля
- `value` (str) - значение (только для hset)

### r.hgetall()
**Что делает:** Получает все поля и значения hash  
**Вход:**
- `key` (str) - ключ hash  
**Выход:** словарь {field: value}

### r.sadd() / r.smembers()
**Что делает:** Добавляет в set / получает все элементы set  
**Вход:**
- `key` (str) - ключ set
- `*values` - значения для добавления (только для sadd)

### r.publish()
**Что делает:** Публикует сообщение в канал pub/sub  
**Вход:**
- `channel` (str) - имя канала
- `message` (str) - сообщение

### r.flushdb()
**Что делает:** Удаляет все ключи в текущей базе данных  
**Вход:** нет
